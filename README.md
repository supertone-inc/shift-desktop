### Installation
1. Go to the [Releases](https://github.com/supertone-inc/shift-desktop/releases) page
2. Download the appropriate installer for your operating system:
   - Windows: `.exe` installer
   - macOS: `.dmg` package
3. Run the installer and follow the setup instructions
4. Launch Supertone Shift from your applications menu

## Getting Started

1. Launch the application
2. Sign in with your Supertone account or create a new one
3. Start exploring the features and capabilities of Supertone Shift
